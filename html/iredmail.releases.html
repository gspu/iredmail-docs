iRedMail: release notes and upgrade tutorials

Usually, upgrading iRedMail is just updating some config files to achieve new features or fix bugs, you do NOT need to download the latest iRedMail and run the script.

Note: Upgrades are only supported from one release to the release immediately following it. Do NOT skip releases.

If you're looking for upgrading a very old iRedMail server to a new one, there's an alternative solution: Migrating to a new iRedMail server.

You can always download the latest iRedMail release on Download page.
